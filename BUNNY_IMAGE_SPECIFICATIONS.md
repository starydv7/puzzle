## üìê **TECHNICAL DETAILS**

### File Naming
- Use lowercase with hyphens: `bunny-happy.png`
- No spaces or special characters
- Be consistent

### Image Quality
- **Resolution**: At least 2x for retina displays
  - If displaying at 200px, create 400px image
  - React Native will scale automatically
- **File Size**: Optimize but keep quality
  - PNG: Use tools like TinyPNG
  - Keep under 200KB per image when possible

### Transparency
- Use PNG with alpha channel
- No white backgrounds (unless intentional)
- Soft edges (anti-aliasing)

### Color Consistency
- Use same color palette throughout
- Pastel, soft colors
- Not too bright or harsh
- Kid-friendly palette

---
